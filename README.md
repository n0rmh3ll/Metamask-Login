
# Metamask Login

A Simple metamask login page


## Run Locally

Clone the project

```bash
  https://github.com/n0rmh3ll/Metamask-Login.git
```

Go to the project directory

```bash
  cd Metamask-Login
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
